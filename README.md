## ğŸš€ Getting Started

### Running Locally

1. Clone the repository
```bash
git clone https://github.com/Hussein5645/AIAS-BASRA-WEBSITE.git
```

2. Navigate to the project directory
```bash
cd AIAS-BASRA-WEBSITE
```

3. Open with a local server (recommended)
```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

4. Open your browser and navigate to `http://localhost:8000`

### Direct File Opening

You can also open `index.html` directly in your browser, but a local server is recommended for the best experience.
