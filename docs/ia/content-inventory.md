# Content Inventory - AIAS Basra Website

## Purpose
This document catalogs all existing pages, posts, and products on the AIAS Basra website to establish a comprehensive understanding of the current content structure before redesign.

## Current Pages Inventory

### Primary Navigation Pages
1. **Home** (`index.html`)
   - Purpose: Landing page with hero, mission, quick links
   - Key Content: Welcome message, mission cards, explore links
   - Priority: High (Entry point)

2. **Events** (`events.html`)
   - Purpose: Display upcoming and past chapter events
   - Key Content: Event cards, registration links, dates
   - Priority: High (Core member engagement)

3. **Education** (`education.html`)

   - Key Content: Program cards, course information
   - Priority: High (Core value proposition)

4. **Magazine** (`magazine.html`)

   - Priority: Medium (Content distribution)

5. **Library** (`library.html`)
   - Purpose: Resource repository
   - Key Content: Downloadable resources, categories, search
   - Priority: High (Resource hub)

6. **About Us** (`about.html`)
   - Purpose: Chapter information and team
   - Key Content: History, values, team members
   - Priority: Medium (Informational)
